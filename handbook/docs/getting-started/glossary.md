# Glossary & acronyms

The vocabulary of the SCEN coordinator role — French academic terms, SUAD
systems, and staffing categories.

## Programmes & levels

| Term | Meaning |
|---|---|
| **FYS** | Foundation Year in Sciences — the pre-Bachelor year. In S1 the Maths and Physics tracks are the same; students may switch track after S1. |
| **FLE / FY** | Foundation-level programmes (FLE = Foundation Langue Étrangère; FY = Foundation Year). |
| **L1 / L2 / L3** | Licence (Bachelor) years 1–3. |
| **M1 / M2** | Master years 1–2. |
| **BSc Maths / BSc Physics** | The two SCEN Bachelor programmes (L1–L3). |
| **MIAI** | Master in Management of Innovation and Artificial Intelligence (opens Sept 2026). |
| **Maquette** | The programme structure / curriculum (courses, hours, credits). Maintained with the HoD; a *maquette change* means courses/codes change and CRNs must be recreated. |
| **Barème** | The grading scale / marking scheme for an assessment (e.g. out of 20, 80, 100). |

## Grades & assessment

| Term | Meaning |
|---|---|
| **PV** (procès-verbal) | The official grade record containing **all** students' grades for a programme/semester. Signed and archived. |
| **Transcript** (relevé de notes, **RDN** / **RN**) | An **individual** student's grade record. |
| **Jury** | The grade validation committee meeting that ratifies results and progression decisions. Held per programme/level after grades are in. |
| **Session 1 / Session 2** | Session 1 = the first exam sitting. Session 2 = the **catch-up (resit)** sitting. |
| **CC** (contrôle continu) | Continuous assessment. At SUAD **all final exams are recorded as CC** — there is no separate *contrôle final*. |
| **CM / TD / TP** | *Cours magistral* (lecture) / *Travaux dirigés* (tutorial) / *Travaux pratiques* (practical / lab). Course hours are counted per type. |
| **Fundamental Block** | A block of courses a student must validate. For L2 (from 26-27) students must validate the semester **and** the fundamental block. |
| **Refus de compensation** | A student's formal refusal of grade *compensation* for a semester — used when they want to sit a resit rather than pass by compensation. Does **not** apply to FYS. See [the procedure](../procedures/refus-de-compensation.md). |
| **Compensation** | Passing a semester on the aggregate/average even if an individual course is below the pass mark. |
| **Validation** | Passing (a course/semester/year) at the required threshold (typically 50/100 for a course). |
| **Catch-up (rattrapage)** | The resit exam session for courses a student did not validate. |

## Staffing categories

| Term | Meaning |
|---|---|
| **FT** | Full-time faculty. HR handles their data entry; coordinators only enter them in the timetable and assign them to courses on Portal. **Not** the coordinator's jurisdiction for requisitions. |
| **PT** | Part-time faculty. Coordinators handle requisitions, CID (if new), and timesheets. |
| **VP** | Visiting Professor. Onboarded via SIMAS + the VP Onboarding app. |
| **PROA** | Professional on Assignment. **Local PROA** = no PhD; **Flying PROA** = coming from abroad. Created via SIMAS. |
| **MDC** (maître de conférences) | Lecturer rank — less research experience than a professor; paid a lower hourly rate (~€400/hr vs ~€500/hr for a professor). The HoD decides the rank. |
| **TA** | Teaching Assistant — an interim contract basis (e.g. AED 150/hr, rising to AED 350/hr on proof of a master's degree). |
| **CID** | The security **clearance** required to hire and pay a new part-timer / VP. Requires a document pack (see [Staffing](../procedures/staffing-contracts.md)) and is processed with HR (Mona Mohamed). |

## Systems & bodies

| Term | Meaning |
|---|---|
| **Banner** | Ellucian Student Information System. Courses (`SCACRSE`), sections/CRNs (`SSASECT`), FYS grades (`SHATCKN`). |
| **CRN** | Course Reference Number — a specific offered section of a course in a term. |
| **Parent CRN** | A grouping CRN used to aggregate/relate sections (e.g. for communication or roll-up). |
| **Portal** | Registrar Portal — reads from Banner; student lists; assign professors (*Faculty Services → Monitoring Courses*). |
| **Blackboard** | Teaching & assessment platform (= Moodle-style). |
| **SRS / Argos** | Reporting tool to download FYS transcripts. |
| **SHATCKN** | Banner screen to upload/change **FYS** student grades. |
| **SSASECT / SCACRSE** | Banner screens: create CRN / create course. |
| **SIMAS** | Registrar system to create Banner IDs for VP / PROA. |
| **Apogée** | Sorbonne Paris grade/records system. |
| **HEDB** | Government reporting interface ("sheds" of old) that reads from Banner. |
| **Serco** | Facilities-management contractor; runs timetabling/scheduling (via Sonali; tool: Evenmaps/Optim solutions). |
| **ADERP / Oracle Fusion / EBS** | HR/ERP self-service and back-office systems. |
| **ServiceHub** | IT/facilities ticketing (`esm.gov.ae/servicehub`). |
| **CAA** | The accreditation authority; reviews course content (e.g. on Blackboard). |
| **DC** | Disciplinary Committee (academic-integrity cases). See [the procedure](../procedures/disciplinary-committee.md). |
| **MoM** | Minutes of Meeting — e.g. the Disciplinary Committee decision document. |
| **PR** | Purchase Requisition (procurement, e.g. VP flights/hotels, lab equipment). |
| **GL** | General Ledger — accounting combination codes used on PRs. |
